## üöÄ Getting Started

### Prerequisites

- Node.js 18+
- Yarn package manager

### Installation

1. **Clone the repository**
```bash
   git clone https://github.com/brenobarbosa01/kits-club.git
   cd kits-club
```

2. **Install dependencies**
```bash
yarn install
```

3. **Run the development server**
```bash
yarn dev
```
